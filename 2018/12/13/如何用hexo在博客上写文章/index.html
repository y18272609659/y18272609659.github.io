如何用hexo在博客上写文章

December 13, 2018 本文总阅读量次

首先，在blog文档下右键启动 Git Bash 输入：

$ hexo n “文章标题”

然后在 blog\source_posts 文件夹中会出现一个新的md文件，打开编写文章即可。

（如果直接在_posts文件夹下新建md文件，效果和上面是一样的）

文章编写完成后，执行在 git bash 中执行命令，上传文件：

$ hexo g

$ hexo d

（上面两行命令可以简写为一行： $ hexo d -g ）
