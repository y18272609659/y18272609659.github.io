Git stash 指令

在使用git对代码进行版本管理的时候，git stash指令是一个非常方便且人性化的指令。如果你在某一分支上编写代码时，突然需要切换到其他分支上去，stash可以帮忙储藏当前分支上未commit的文件，分支干净了，当然也就可以切换分支啦。

储藏指令：
$ git stash

查看现有储藏：
$ git stash list

stash@{0}: xxx
stash@{1}: xxxxx
stash@{2}: xxxxxx

拿出最近的储藏：
$ git stash apply

拿出更早的储藏：
$ git stash apply stash@{2}

拿出最近的储藏，并将其从list中移除：
$ git stash pop

（还有几个指令还没完全弄清楚，官方stash文档）
