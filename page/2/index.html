Git stash 指令

一月 12, 2019

在使用git对代码进行版本管理的时候，git stash指令是一个非常方便且人性化的指令。如果你在某一分支上编写代码时，突然需要切换到其他分支上去，stash可以帮忙储藏当前分支上未commit的文件，分支干净了，当然也就可以切换分支啦。

储藏指令：
$ git stash

查看现有储藏：
$ git stash list

stash@{0}: xxx
stash@{1}: xxxxx
stash@{2}: xxxxxx

拿出最近的储藏：
$ git stash apply

拿出更早的储藏：
$ git stash apply stash@{2}

拿出最近的储藏，并将其从list中移除：
$ git stash pop

（还有几个指令还没完全弄清楚，官方stash文档）

如何用hexo在博客上写文章

十二月 13, 2018

首先，在blog文档下右键启动 Git Bash 输入：

$ hexo n “文章标题”

然后在 blog\source_posts 文件夹中会出现一个新的md文件，打开编写文章即可。

（如果直接在_posts文件夹下新建md文件，效果和上面是一样的）

文章编写完成后，执行在 git bash 中执行命令，上传文件：

$ hexo g

$ hexo d

（上面两行命令可以简写为一行： $ hexo d -g ）
